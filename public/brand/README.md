# Sentinel — brand assets

The mark is **the eye that watches your browser** — bold hand-drawn line-art
with a single spark. Two colorways: black on paper, and inverted (cream spark
and all) for dark UI.

## Files

| File | Use |
|------|-----|
| `sentinel-eye.svg` | Primary mark, black `#111`. Scalable, no font dependency. |
| `sentinel-eye-dark.svg` | Inverted — cream `#f3f1ea`, transparent bg. For dark surfaces. |
| `sentinel-favicon.svg` | Bare eye (no spark), black. Use at ≤32px. |
| `sentinel-favicon-dark.svg` | Bare eye, cream. |
| `png/sentinel-eye-512.png` | Raster master, black, transparent. GitHub avatar / social. |
| `png/sentinel-eye-dark-512.png` | Raster master, inverted, transparent. |
| `png/sentinel-favicon-180.png` | Apple touch icon. |
| `png/sentinel-favicon-32.png` | Favicon fallback. |

## Wordmark

`Sentinel`, set in **Shantell Sans** 700 (open-licensed, Google Fonts) — a
casual marker hand that matches the drawn eye. The lockup is the mark SVG
placed beside live text; bake to PNG if you need it font-free.

```html
<link rel="icon" href="/brand/sentinel-favicon.svg">
<link rel="apple-touch-icon" href="/brand/png/sentinel-favicon-180.png">
```

## Palette

- Ink `#111111` · Paper `#fdfdfb` · Dark `#141414` · Cream on dark `#f3f1ea`

## Don't

Add gradients, shadows, or a second accent color. Taper the strokes. Recolor
the eye outline. Put it in a rounded box. Keep clear space ≈ the pupil height.
